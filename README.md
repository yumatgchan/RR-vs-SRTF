# RR-vs-SRTF

# PROJECT DESCRIPTION
explains RR and SRTF in a comparison way.
this project includes comparison table of both algorithms and their calculations metrics. (WT,TAT,RT)
it also describes all the required interface to facilite its usage on the user.

# REQUIREMENTS
Python 3.8+ version, Tkinter and a linux installation note 
"python -m src.gui.main_ui" without the equation marks inside the terminal

# BUILD/ RUN STEPS
1 - VS code
2 - open folder select SRC or the whole project
3 - run the gui file
4 - write in terminal -> python -m src.gui.main_ui
5 - fill the data or load a scenario
6 - run the program or reset the data
